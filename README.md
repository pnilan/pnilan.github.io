pnilan.github.io
================

Personal portfolio site. Can be seen at [pnilan.github.io](http://pnilan.github.io).

Used [Foundation](http://foundation.zurb.com/), [Font Awesome](http://fortawesome.github.io/Font-Awesome/), [Stellar.js](http://markdalgleish.com/projects/stellar.js/) for parallax scrolling, [AnimateScroll](http://plugins.compzets.com/animatescroll/) for automatic, easing scrolling, and [jQuery Transit](http://ricostacruz.com/jquery.transit/) for simple animations.
